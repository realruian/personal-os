首次使用：贴入你的 GitHub Personal Access Token，存到这个浏览器。

· Token 只存在你当前浏览器的 localStorage，不会上传到任何服务器
· 需要 fine-grained PAT，repo 限定 personal-os，权限 Contents: Read and write
· 丢了就去 GitHub 重建

最多显示前 5 篇（按时间倒序）。改时间戳可调顺序。
