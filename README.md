## 这个网站上有什么

- **一栏身份**：简单介绍我是谁，以及怎么联系我
- **一栏想法**：我每天读到、想到、做到的一些片段
- **一栏图片**：一些我觉得值得留下来的视觉记录

以及一点点氛围 —— 按 `D` 切白天，按 `N` 回到夜晚。
`S` 有飘叶和森林声，`M` 有月色，`R` 有雨。
想看点好玩的就按 `C`。

右上角可以切中英文。移动端请点左上角 logo 的四个字母。
